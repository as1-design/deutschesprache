# Deutsche Sprache — Learning Tools

A small, growing collection of self-contained, single-file HTML tools for German language learning and speaking practice, hosted on GitHub Pages.

🔗 **Live site:** https://as1-design.github.io/deutschesprache/

---

## About

Each tool is a standalone `.html` file — no build step, no dependencies, no server. Open it in a browser (or visit the live link above) and it just works, including on mobile.

## Tools

| Tool | Type | Level | Description |
|------|------|-------|-------------|
| [🃏 German Verb Memory Cards](german-memory-cards_basic%20verbs-opposites.html) | Print & play game | A2–B1 | Print-and-cut memory-matching game built around 28 verb/opposite pairs (*öffnen ↔ schließen*, *kommen ↔ gehen*). One-click A4 print layout (double-sided: fronts then backs), plus an on-screen preview. |
| [🗣️ Anekdoten-Board](german-speaking-board_b2.1_anecdote.html) | Speaking prompts | B2.1 | Digital prompt board for anecdote-style speaking practice. 12 topic categories × 5 questions (60 prompts), a random-question shuffle, and a 60-second countdown ring timer with pause/reset. |
| [📖 Sprechreference](sprechreference_b2.1_deutsch.html) | Speaking reference | B2.1 | Mobile-first phrasebank for the oral exam: opinions, suggestions and turn-taking. Tap a phrase to mark it *used* ✓ or *want to try* ★; marks are saved on your device. Formal/informal (Sie/du) and Konjunktiv II noted throughout. |
